# How to Create GitHub Repository - Step by Step

## 🎯 Quick Steps (5 minutes)

### Step 1: Go to GitHub
1. Open your browser
2. Go to: **https://github.com/new**
3. Make sure you're logged in as **yeswanthsetti-tech**

### Step 2: Fill in Repository Details
1. **Repository name**: Type `tts-app` (or any name you like)
2. **Description** (optional): "Text-to-Speech application with CLI and Streamlit web app"
3. **Visibility**: 
   - Choose **Public** (recommended - free for Streamlit hosting)
   - OR **Private** (if you want it private)
4. **⚠️ IMPORTANT**: 
   - **DO NOT** check ✅ "Add a README file"
   - **DO NOT** check ✅ "Add .gitignore" 
   - **DO NOT** check ✅ "Choose a license"
   - Leave all checkboxes **UNCHECKED** ❌

### Step 3: Create Repository
1. Click the green **"Create repository"** button at the bottom

### Step 4: After Creating
1. GitHub will show you a page with instructions
2. **DON'T follow those instructions** (they're for empty repos)
3. **Just tell me**: "I created the repository" or "Done"
4. I will push your code for you!
